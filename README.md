# DragonThrone Ultimate — GOT + HOTD Enterprise Fan App

This project extends the original `GOT-dragon-dynasty-quiz-upgrade` project instead of replacing it.

## What is included

- Original GOT project assets preserved
- Original GOT video preserved at `public/video/one.mp4`
- New HOTD cinematic video generated at `public/video/hotd.mp4`
- Separate tabs for Game of Thrones and House of the Dragon
- Fullscreen cinematic video hero
- Global search across characters, timeline and map locations
- GOT timeline and HOTD timeline
- Houses section for each universe
- Character cards with photos, quotes, status and story tags
- HOTD dragon gallery
- Interactive Westeros / Dragonstone style map with clickable points
- Quiz entry section
- Enterprise-style responsive UI

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

This is a fan-made UI concept. Images and video are placed in `public/images` and `public/video`. Replace placeholder/cropped character photos with licensed production stills if you plan to publish.
